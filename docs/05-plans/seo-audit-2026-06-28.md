# SEO audit — 2026-06-28

**Host:** `https://arcane-reader.vercel.app`  
**Scope:** Implementation of findings from SEO audit plan (code + docs). GSC remains human-ops.

## Summary

| Area              | Before                                 | After (code)                                                      |
| ----------------- | -------------------------------------- | ----------------------------------------------------------------- |
| robots.txt parity | Vercel missing `/translation-requests` | `buildRobotsTxt` SSOT in `src/shared/robotsTxt.ts`                |
| `/account-tiers`  | Default SPA meta                       | SSR + sitemap + vercel rewrite                                    |
| `/news/:slug`     | Default SPA meta                       | SSR + NewsArticle JSON-LD + sitemap + `usePageMeta`               |
| Client SPA meta   | Publications only                      | `useStaticPageMeta` for info pages; news detail via `usePageMeta` |
| Title suffix      | `Новости — Arcane \| Arcane`           | `staticPageDocumentTitle` skips duplicate suffix                  |
| twitter:image     | Missing                                | SSR injectors + `index.html` + client hooks                       |

## Findings resolved in this sprint

### Finding: robots.txt Vercel / Express mismatch

- Severity: Warning → **Fixed**
- Fix: `src/shared/robotsTxt.ts`; `api/robots.ts` and `sendRobotsTxt` use `buildRobotsTxt(getPublicBaseUrl(req))`
- Owner: API

### Finding: `/news/:slug` without crawler meta

- Severity: Warning → **Fixed**
- Fix: `serveNewsDetailHtml`, vercel rewrite `/news/:slug*`, sitemap news URLs (`SITEMAP_NEWS_LIMIT=100`), `NewsArticle` JSON-LD, `usePageMeta` on `NewsDetailPage`
- Owner: API + UI

### Finding: `/account-tiers` meta prepared but not served

- Severity: Info → **Fixed**
- Fix: `STATIC_SEO_PATHS`, vercel rewrite, sitemap static list
- Owner: API

### Finding: SPA navigation stale meta on info pages

- Severity: Info → **Fixed**
- Fix: `useStaticPageMeta` on About, Contact, Privacy, Terms, News list, AccountTiers
- Owner: UI

### Finding: Double title suffix on static SSR pages

- Severity: Info → **Fixed**
- Fix: `staticPageDocumentTitle()` in `src/shared/staticPageMeta.ts`
- Owner: API

### Finding: Missing twitter:image

- Severity: Info → **Fixed**
- Fix: `injectPublicationMeta`, `injectStaticPageMeta`, `index.html`, `usePageMeta`, `useStaticPageMeta`
- Owner: API + UI
