# 3.2 РЎРѕРіР»Р°СЃРѕРІР°РЅРЅРѕРµ СЂР°Р·Р±РёРµРЅРёРµ РІ Stage 3 вЂ” Р°РЅР°Р»РёР·

## РўРµРєСѓС‰РµРµ РїРѕРІРµРґРµРЅРёРµ

### Stage 2 (РїРµСЂРµРІРѕРґ)

- Р’С…РѕРґ: `sourceText` (РѕСЂРёРіРёРЅР°Р»).
- Р Р°Р·Р±РёРµРЅРёРµ: `chunkText(sourceText, { maxTokens, preserveParagraphs, neverSplitParagraphs })` вЂ” РіСЂР°РЅРёС†С‹ РїРѕ РїР°СЂР°РіСЂР°С„Р°Рј РѕСЂРёРіРёРЅР°Р»Р°.
- Р РµР·СѓР»СЊС‚Р°С‚: РјР°СЃСЃРёРІ `chunkResults[]` СЃ РїР°СЂР°РјРё `{ original, translated }` Рё РїРѕР»РЅС‹Р№ `translatedText = mergeChunks(chunkResults)` (СЃРєР»РµР№РєР° С‡РµСЂРµР· `\n\n`).
- РС‚РѕРі: **РѕРґРЅРѕР·РЅР°С‡РЅРѕРµ СЃРѕРѕС‚РІРµС‚СЃС‚РІРёРµ** В«С‡Р°РЅРє РѕСЂРёРіРёРЅР°Р»Р° i в†” С‡Р°РЅРє РїРµСЂРµРІРѕРґР° iВ».

### Stage 3 (СЂРµРґР°РєС‚СѓСЂР°)

- Р’С…РѕРґ: РїРѕР»РЅС‹Рµ СЃС‚СЂРѕРєРё `translatedText` Рё `originalText`.
- Р Р°Р·Р±РёРµРЅРёРµ: **РЅРµР·Р°РІРёСЃРёРјРѕ** РґР»СЏ РєР°Р¶РґРѕРіРѕ С‚РµРєСЃС‚Р°:
  - `translatedChunks = chunkText(translatedText, { maxTokens, preserveParagraphs })`
  - `originalChunks = chunkText(originalText, { maxTokens, preserveParagraphs })`
- Р РµРґР°РєС‚РёСЂРѕРІР°РЅРёРµ: РґР»СЏ РєР°Р¶РґРѕРіРѕ РёРЅРґРµРєСЃР° `i` РІС‹Р·С‹РІР°РµС‚СЃСЏ `editChunk(translatedChunks[i], originalChunks[i])`.
- РЎРєР»РµР№РєР°: `mergeChunks(editedChunks)`.

### Р’ С‡С‘Рј РїСЂРѕР±Р»РµРјР°

1. **Р Р°Р·РЅРѕРµ С‡РёСЃР»Рѕ С‡Р°РЅРєРѕРІ**  
   РћСЂРёРіРёРЅР°Р» Рё РїРµСЂРµРІРѕРґ СЂРµР¶СѓС‚СЃСЏ РїРѕ СЃРІРѕРёРј РїР°СЂР°РіСЂР°С„Р°Рј Рё Р»РёРјРёС‚Сѓ С‚РѕРєРµРЅРѕРІ. Р’ РїРµСЂРµРІРѕРґРµ РїР°СЂР°РіСЂР°С„С‹ РјРѕРіСѓС‚ Р±С‹С‚СЊ РґР»РёРЅРЅРµРµ/РєРѕСЂРѕС‡Рµ, РѕР±СЉРµРґРёРЅРµРЅС‹ РёР»Рё СЂР°Р·Р±РёС‚С‹ РјРѕРґРµР»СЊСЋ. Р’ РёС‚РѕРіРµ:
   - `translatedChunks.length` РјРѕР¶РµС‚ РЅРµ СЃРѕРІРїР°РґР°С‚СЊ СЃ `originalChunks.length`;
   - РїСЂРё `i >= originalChunks.length` РёСЃРїРѕР»СЊР·СѓРµС‚СЃСЏ `originalChunks[i] || { content: '', index: i }` вЂ” Рє РїРµСЂРµРІРѕРґСѓ С‡Р°РЅРєР° i РїСЂРёРєР»Р°РґС‹РІР°РµС‚СЃСЏ РїСѓСЃС‚РѕР№ РёР»Рё РЅРµРІРµСЂРЅС‹Р№ РєСѓСЃРѕРє РѕСЂРёРіРёРЅР°Р»Р°.

2. **РЎРґРІРёРі РіСЂР°РЅРёС†**  
   Р”Р°Р¶Рµ РїСЂРё РѕРґРёРЅР°РєРѕРІРѕРј С‡РёСЃР»Рµ РїР°СЂР°РіСЂР°С„РѕРІ РіСЂР°РЅРёС†С‹ С‡Р°РЅРєРѕРІ РїРѕ С‚РѕРєРµРЅР°Рј РІ РѕСЂРёРіРёРЅР°Р»Рµ Рё РІ РїРµСЂРµРІРѕРґРµ РЅРµ СЃРѕРІРїР°РґР°СЋС‚: РІ РѕРґРёРЅ В«С‡Р°РЅРєВ» РѕСЂРёРіРёРЅР°Р»Р° РјРѕР¶РµС‚ РїРѕРїР°СЃС‚СЊ 3 Р°Р±Р·Р°С†Р°, РІ РѕРґРёРЅ С‡Р°РЅРє РїРµСЂРµРІРѕРґР° вЂ” 2. РўРѕРіРґР° `translatedChunks[i]` Рё `originalChunks[i]` СЃРµРјР°РЅС‚РёС‡РµСЃРєРё РЅРµ СЃРѕРѕС‚РІРµС‚СЃС‚РІСѓСЋС‚ РґСЂСѓРі РґСЂСѓРіСѓ.

3. **Р РµР·СѓР»СЊС‚Р°С‚**  
   Р РµРґР°РєС‚РѕСЂ РїРѕР»СѓС‡Р°РµС‚ РїР°СЂСѓ (РѕСЂРёРіРёРЅР°Р», РїРµСЂРµРІРѕРґ), РЅРµ РІС‹СЂРѕРІРЅРµРЅРЅСѓСЋ РїРѕ СЃРјС‹СЃР»Сѓ в†’ С…СѓР¶Рµ РєРѕРЅС‚РµРєСЃС‚ РґР»СЏ СЂРµРґР°РєС‚СѓСЂС‹, СЂРёСЃРє РёСЃРїРѕСЂС‚РёС‚СЊ СЃС‚С‹РєРё РїСЂРё СЃРєР»РµР№РєРµ.

## Р§С‚Рѕ РјРѕР¶РЅРѕ СЃРґРµР»Р°С‚СЊ

### Р’Р°СЂРёР°РЅС‚ A: РСЃРїРѕР»СЊР·РѕРІР°С‚СЊ РїР°СЂС‹ С‡Р°РЅРєРѕРІ РёР· Stage 2 (СЂРµРєРѕРјРµРЅРґСѓРµС‚СЃСЏ РґР»СЏ РїРѕР»РЅРѕРіРѕ РїР°Р№РїР»Р°Р№РЅР°)

**РРґРµСЏ:** РєРѕРіРґР° Stage 3 Р·Р°РїСѓСЃРєР°РµС‚СЃСЏ СЃСЂР°Р·Сѓ РїРѕСЃР»Рµ Stage 2 РІ РѕРґРЅРѕРј РїР°Р№РїР»Р°Р№РЅРµ, РЅРµ СЂРµР·Р°С‚СЊ С‚РµРєСЃС‚ Р·Р°РЅРѕРІРѕ, Р° РїРµСЂРµРґР°С‚СЊ РІ Stage 3 СѓР¶Рµ РіРѕС‚РѕРІС‹Рµ РїР°СЂС‹ `(original, translated)` РёР· `stage2Result.data.chunkResults`.

**РџР»СЋСЃС‹:**

- РњРёРЅРёРјСѓРј РёР·РјРµРЅРµРЅРёР№: Stage 2 СѓР¶Рµ РѕС‚РґР°С‘С‚ `chunkResults[]` СЃ `original` Рё `translated`.

**Р РµР°Р»РёР·Р°С†РёСЏ (РєРѕРЅС†РµРїС‚):**

   - РµСЃР»Рё РµСЃС‚СЊ `stage2Result.data.chunkResults`, СЃРѕР±СЂР°С‚СЊ РїР°СЂС‹

     Рё РІС‹Р·РІР°С‚СЊ Stage 3 СЃ СЌС‚РёРјРё РїР°СЂР°РјРё;
   - РёРЅР°С‡Рµ (РЅР°РїСЂРёРјРµСЂ, С‚РѕР»СЊРєРѕ СЂРµРґР°РєС‚СѓСЂР° Р±РµР· РїРµСЂРµРІРѕРґР°) вЂ” РѕСЃС‚Р°РІРёС‚СЊ С‚РµРєСѓС‰СѓСЋ Р»РѕРіРёРєСѓ: РїРµСЂРµРґР°РІР°С‚СЊ РїРѕР»РЅС‹Рµ `translatedText` Рё `originalText`, Stage 3 СЂРµР¶РµС‚ РѕР±Р° С‚РµРєСЃС‚Р° СЃР°Рј.

### Р’Р°СЂРёР°РЅС‚ B: РҐСЂР°РЅРёС‚СЊ РіСЂР°РЅРёС†С‹ С‡Р°РЅРєРѕРІ Stage 2 РїРѕ СЃРёРјРІРѕР»Р°Рј (РѕСЂРёРіРёРЅР°Р»)
