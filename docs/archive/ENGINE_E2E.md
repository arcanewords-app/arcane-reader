   - РµСЃР»Рё РѕРґРёРЅ РїР°СЂР°РіСЂР°С„ СѓР¶Рµ Р±РѕР»СЊС€Рµ `maxTokens`, РѕРЅ СЂРµР¶РµС‚СЃСЏ РїРѕ РїСЂРµРґР»РѕР¶РµРЅРёСЏРј (`chunkBySentences`) Рё СѓС…РѕРґРёС‚ РІ РЅРµСЃРєРѕР»СЊРєРѕ С‡Р°РЅРєРѕРІ;
   - РёРЅР°С‡Рµ РїР°СЂР°РіСЂР°С„С‹ РїРѕРґСЂСЏРґ СЃРєР»РµРёРІР°СЋС‚СЃСЏ С‡РµСЂРµР· `\n\n`, РїРѕРєР° РЅРµ РїРµСЂРµРїРѕР»РЅСЏС‚ Р»РёРјРёС‚ вЂ” С‚РѕРіРґР° РЅР°С‡РёРЅР°РµС‚СЃСЏ РЅРѕРІС‹Р№ С‡Р°РЅРє.
3. РЈ РєР°Р¶РґРѕРіРѕ С‡Р°РЅРєР°: `id: "chunk_0"`, `"chunk_1"`, вЂ¦; `content` (СЃС‚СЂРѕРєР°); `index` (0, 1, 2, вЂ¦); `tokenCount`.

**Р РµР¶РёРј РїРѕ РїСЂРµРґР»РѕР¶РµРЅРёСЏРј** (РїСЂРё `preserveParagraphs: false` РёР»Рё РІРЅСѓС‚СЂРё СЂР°Р·Р±РёРµРЅРёСЏ Р±РѕР»СЊС€РѕРіРѕ РїР°СЂР°РіСЂР°С„Р°):

- РўРµРєСЃС‚ СЂРµР¶РµС‚СЃСЏ РїРѕ РїР°С‚С‚РµСЂРЅСѓ РєРѕРЅС†Р° РїСЂРµРґР»РѕР¶РµРЅРёСЏ: `(?<=[.!?])\s+(?=[A-ZРђ-РЇРЃ"В«])`.
- Р’ С‡Р°РЅРє РЅР°Р±РёСЂР°СЋС‚СЃСЏ РїСЂРµРґР»РѕР¶РµРЅРёСЏ РґРѕ Р»РёРјРёС‚Р° С‚РѕРєРµРЅРѕРІ; РјРµР¶РґСѓ С‡Р°РЅРєР°РјРё вЂ” РїРµСЂРµРєСЂС‹С‚РёРµ РІ `overlapSentences` РїСЂРµРґР»РѕР¶РµРЅРёР№ (РїСЂРµРґС‹РґСѓС‰РёР№ С…РІРѕСЃС‚ РґСѓР±Р»РёСЂСѓРµС‚СЃСЏ РІ РЅР°С‡Р°Р»Рµ СЃР»РµРґСѓСЋС‰РµРіРѕ С‡Р°РЅРєР°).

**РЎР±РѕСЂ РѕР±СЂР°С‚РЅРѕ** (`mergeChunks`):

- Р’С…РѕРґ: РјР°СЃСЃРёРІ `{ content: string; index: number }[]` (РёР»Рё РѕР±СЉРµРєС‚С‹ СЃ С‚Р°РєРёРјРё РїРѕР»СЏРјРё).
- Р¤РёР»СЊС‚СЂР°С†РёСЏ РїСѓСЃС‚С‹С… `content`, СЃРѕСЂС‚РёСЂРѕРІРєР° РїРѕ `index`, СЃРєР»РµР№РєР°: `sorted.map(c => c.content).join('\n\n')`.
- РџСѓСЃС‚С‹Рµ С‡Р°РЅРєРё РѕС‚Р±СЂР°СЃС‹РІР°СЋС‚СЃСЏ Рё Р»РѕРіРёСЂСѓСЋС‚СЃСЏ; РїРѕСЂСЏРґРѕРє СЃС‚СЂРѕРіРѕ РїРѕ `index`.

**РћРїС†РёСЏ В«РЅРµ РґСЂРѕР±РёС‚СЊ РїР°СЂР°РіСЂР°С„С‹В»** (`neverSplitParagraphs`, РїРѕ СѓРјРѕР»С‡Р°РЅРёСЋ **true**):

- Р•СЃР»Рё РѕРґРёРЅ РїР°СЂР°РіСЂР°С„ РїСЂРµРІС‹С€Р°РµС‚ `maxTokens`, РїСЂРё `neverSplitParagraphs: true` РѕРЅ РІСЃС‘ СЂР°РІРЅРѕ РѕСЃС‚Р°С‘С‚СЃСЏ РѕРґРЅРёРј С‡Р°РЅРєРѕРј (Р»РѕРіРёСЂСѓРµС‚СЃСЏ РїСЂРµРґСѓРїСЂРµР¶РґРµРЅРёРµ). РўР°Рє СЃРѕС…СЂР°РЅСЏРµС‚СЃСЏ 1:1 СЃРѕРѕС‚РІРµС‚СЃС‚РІРёРµ РїР°СЂР°РіСЂР°С„ в†” С‡Р°РЅРє Рё РјРµРЅСЊС€Рµ РѕС€РёР±РѕРє РїСЂРё СЃРёРЅС…СЂРѕРЅРёР·Р°С†РёРё СЃ РїР°СЂР°РіСЂР°С„Р°РјРё.
- РџСЂРё `neverSplitParagraphs: false` (СЃС‚Р°СЂС‹Р№ СЂРµР¶РёРј) СЃР»РёС€РєРѕРј Р±РѕР»СЊС€РѕР№ РїР°СЂР°РіСЂР°С„ СЂРµР¶РµС‚СЃСЏ РїРѕ РїСЂРµРґР»РѕР¶РµРЅРёСЏРј. РџРѕРґСЂРѕР±РЅРµРµ: [CHUNKING_AND_SAVE.md](./CHUNKING_AND_SAVE.md).

### 2.3 Stage 2 (Translation): С‡Р°РЅРєРё Рё СЃР±РѕСЂ

1. **Р Р°Р·Р±РёРµРЅРёРµ**: `chunkText(sourceText, { maxTokens: chunkSize ?? 2000, preserveParagraphs: true })` в†’ РјР°СЃСЃРёРІ `TextChunk[]`.
2. **РћР±СЂР°Р±РѕС‚РєР°**: С†РёРєР» РїРѕ С‡Р°РЅРєР°Рј, РґР»СЏ РєР°Р¶РґРѕРіРѕ:
   - РІС‹Р·РѕРІ `translateChunk(...)` в†’ РѕРґРёРЅ Р·Р°РїСЂРѕСЃ Рє РїСЂРѕРІР°Р№РґРµСЂСѓ;
   - С„РѕСЂРјР°С‚ РѕС‚РІРµС‚Р°: СЃРЅР°С‡Р°Р»Р° РїРѕРїС‹С‚РєР° **JSON** (`completeJSON` СЃ СЃС‚СЂСѓРєС‚СѓСЂРѕР№ `{ paragraphs: [{ id, translated }] }`); РїСЂРё РѕС€РёР±РєРµ РїР°СЂСЃРёРЅРіР° РёР»Рё РѕС‚СЃСѓС‚СЃС‚РІРёРё `completeJSON` вЂ” **fallback РЅР° С‚РµРєСЃС‚** (`complete`, РѕС‚РІРµС‚ РєР°Рє РѕРґРёРЅ С‚РµРєСЃС‚);
   - СЂРµР·СѓР»СЊС‚Р°С‚ РїСѓС€РёС‚СЃСЏ РІ `chunkResults`: `{ chunkId, original, translated }`.
3. **РџСЂРё РѕС€РёР±РєРµ РѕРґРЅРѕРіРѕ С‡Р°РЅРєР°**: РІ РјР°СЃСЃРёРІ РєР»Р°РґС‘С‚СЃСЏ placeholder `{ chunkId, original, translated: "[ERROR: ...]" }`, С†РёРєР» РїСЂРѕРґРѕР»Р¶Р°РµС‚СЃСЏ (РЅРµ РїР°РґР°РµРј РїРѕР»РЅРѕСЃС‚СЊСЋ).
4. **РЎР±РѕСЂ**: РёР· `chunkResults` Р±РµСЂС‘С‚СЃСЏ `chunkId` в†’ РїР°СЂСЃРёС‚СЃСЏ РёРЅРґРµРєСЃ (`chunk_0` в†’ 0); РµСЃР»Рё РёРЅРґРµРєСЃ РЅРµ РёР·РІР»РµС‡С‘РЅ, С‡Р°РЅРє РѕС‚Р±СЂР°СЃС‹РІР°РµС‚СЃСЏ (Р»РѕРіРёСЂСѓРµС‚СЃСЏ РѕС€РёР±РєР°). Р’Р°Р»РёРґРЅС‹Рµ С‡Р°РЅРєРё РїРµСЂРµРґР°СЋС‚СЃСЏ РІ `mergeChunks(validChunks)` в†’ РѕРґРЅР° СЃС‚СЂРѕРєР° `translatedText`.
5. **РџСѓСЃС‚РѕР№ СЂРµР·СѓР»СЊС‚Р°С‚ РїРѕСЃР»Рµ merge**: СЃС‚Р°РґРёСЏ РІРѕР·РІСЂР°С‰Р°РµС‚ `success: false`, pipeline РІ РёС‚РѕРіРµ РѕС‚РґР°С‘С‚ `[ERROR] ...`, СЃРµСЂРІРµСЂ СЃС‚Р°РІРёС‚ РіР»Р°РІСѓ РІ `error`.

**Р¤РѕСЂРјР°С‚ РѕС‚РІРµС‚Р° (РїСѓРЅРєС‚ 1.3 СЂРµС„Р°РєС‚РѕСЂРёРЅРіР°):** РѕСЃРЅРѕРІРЅРѕР№ РїСѓС‚СЊ вЂ” JSON СЃ РїРѕР»РµРј `paragraphs` (СЃРј. TRANSLATOR_SYSTEM_PROMPT); РїСЂРё РѕС€РёР±РєРµ РїР°СЂСЃРёРЅРіР° РёР»Рё РѕС‚СЃСѓС‚СЃС‚РІРёРё completeJSON РёСЃРїРѕР»СЊР·СѓРµС‚СЃСЏ fallback вЂ” РѕР±С‹С‡РЅС‹Р№ С‚РµРєСЃС‚ (Р°Р±Р·Р°С†С‹ Р·Р°С‚РµРј СЂР°Р·Р±РёРІР°СЋС‚СЃСЏ РїРѕ `\n\n` РЅР° СЃРµСЂРІРµСЂРµ РїСЂРё СЃРёРЅРєРµ).

**Fallback РїРѕ С„РѕСЂРјР°С‚Сѓ РѕС‚РІРµС‚Р° (РІРЅСѓС‚СЂРё `translateChunk`):**

- Р•СЃР»Рё РїСЂРѕРІР°Р№РґРµСЂ СѓРјРµРµС‚ `completeJSON` вЂ” Р·Р°РїСЂР°С€РёРІР°РµРј JSON; РїСЂРё Р»СЋР±РѕРј РёСЃРєР»СЋС‡РµРЅРёРё (СЃРµС‚РµРІРѕРј, РїР°СЂСЃРёРЅРіРµ, РЅРµРІРµСЂРЅРѕР№ СЃС‚СЂСѓРєС‚СѓСЂРµ) РїРµСЂРµС…РѕРґРёРј РЅР° `complete` Рё РёСЃРїРѕР»СЊР·СѓРµРј РѕС‚РІРµС‚ РєР°Рє РѕРґРёРЅ С‚РµРєСЃС‚РѕРІС‹Р№ Р±Р»РѕРє.
- Р•СЃР»Рё РїСЂРѕРІР°Р№РґРµСЂ РЅРµ СѓРјРµРµС‚ `completeJSON` вЂ” СЃСЂР°Р·Сѓ РёСЃРїРѕР»СЊР·СѓРµС‚СЃСЏ С‚РѕР»СЊРєРѕ `complete`.

### 2.4 Stage 3 (Editing): С‡Р°РЅРєРё Рё СЃР±РѕСЂ

- **Р РµС€РµРЅРёРµ Рѕ С‡Р°РЅРєРѕРІР°РЅРёРё**: РµСЃР»Рё Р·Р°РґР°РЅ `chunkSize` РёР»Рё РѕС†РµРЅРєР° С‚РѕРєРµРЅРѕРІ С‚РµРєСЃС‚Р° > 3000, РёСЃРїРѕР»СЊР·СѓРµС‚СЃСЏ С‡Р°РЅРєРѕРІР°РЅРЅР°СЏ СЂРµРґР°РєС‚СѓСЂР° (`editChunked`).
- **Р Р°Р·Р±РёРµРЅРёРµ**: Рё РїРµСЂРµРІРµРґС‘РЅРЅС‹Р№, Рё РѕСЂРёРіРёРЅР°Р»СЊРЅС‹Р№ С‚РµРєСЃС‚ СЂРµР¶СѓС‚СЃСЏ РѕРґРЅРёРј Рё С‚РµРј Р¶Рµ `chunkText(..., preserveParagraphs: true)` в†’ РїРѕР»СѓС‡Р°РµРј РґРІР° РјР°СЃСЃРёРІР° С‡Р°РЅРєРѕРІ; СЂРµРґР°РєС‚СѓСЂР° РёРґС‘С‚ РїРѕ РїР°СЂР°Рј `translatedChunks[i]` Рё `originalChunks[i]` (РїРѕ РёРЅРґРµРєСЃСѓ).
- **РћР±СЂР°Р±РѕС‚РєР°**: РґР»СЏ РєР°Р¶РґРѕР№ РїР°СЂС‹ РІС‹Р·РѕРІ `editChunk(...)` в†’ РѕРґРёРЅ `complete`; СЂРµР·СѓР»СЊС‚Р°С‚ (РёР»Рё РёСЃС…РѕРґРЅС‹Р№ С‚РµРєСЃС‚ С‡Р°РЅРєР° РїСЂРё РѕС€РёР±РєРµ) РїСѓС€РёС‚СЃСЏ РІ `editedChunks` СЃ С‚РµРј Р¶Рµ `index`.
- **РџСЂРё РѕС€РёР±РєРµ С‡Р°РЅРєР°**: РІ РјР°СЃСЃРёРІ РєР»Р°РґС‘С‚СЃСЏ РёСЃС…РѕРґРЅС‹Р№ РїРµСЂРµРІРµРґС‘РЅРЅС‹Р№ С‡Р°РЅРє (`translatedChunk.content`) вЂ” СЂРµРґР°РєС‚СѓСЂР° СЌС‚РѕРіРѕ РєСѓСЃРєР° РїСЂРѕРїСѓСЃРєР°РµС‚СЃСЏ, РїРѕСЂСЏРґРѕРє СЃРѕС…СЂР°РЅСЏРµС‚СЃСЏ.
- **РЎР±РѕСЂ**: `mergeChunks(editedChunks)` в†’ РѕРґРЅР° СЃС‚СЂРѕРєР°. Р•СЃР»Рё РїРѕСЃР»Рµ merge СЃС‚СЂРѕРєР° РїСѓСЃС‚Р°СЏ вЂ” РІРѕР·РІСЂР°С‰Р°РµРј РёСЃС…РѕРґРЅС‹Р№ `translatedText` С†РµР»РёРєРѕРј Рё СЃРїРёСЃР°РЅРЅС‹Рµ С‚РѕРєРµРЅС‹ (fallback В«РѕСЃС‚Р°РІРёС‚СЊ РєР°Рє Р±С‹Р»РѕВ»).

**Quality check**: РїСЂРё `checkQuality: true` Рё **РЅРµ** С‡Р°РЅРєРѕРІР°РЅРЅРѕР№ СЂРµРґР°РєС‚СѓСЂРµ РІС‹Р·С‹РІР°РµС‚СЃСЏ РѕС‚РґРµР»СЊРЅС‹Р№ Р·Р°РїСЂРѕСЃ `completeJSON` РґР»СЏ РѕС†РµРЅРєРё РєР°С‡РµСЃС‚РІР°; РїСЂРё С‡Р°РЅРєРѕРІР°РЅРЅРѕР№ СЂРµРґР°РєС‚СѓСЂРµ quality check РЅРµ РґРµР»Р°РµС‚СЃСЏ (С‡С‚РѕР±С‹ РЅРµ СЂР°Р·РґСѓРІР°С‚СЊ РІСЂРµРјСЏ Рё С‚Р°Р№РјР°СѓС‚С‹).
