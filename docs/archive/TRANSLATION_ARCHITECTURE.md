## 1. РџСЂРёРЅС†РёРїС‹

### 1.1 РћРґРЅР° РѕРїРµСЂР°С†РёСЏ вЂ” В«РїРµСЂРµРІРѕРґ РіР»Р°РІС‹В»

- **РџРµСЂРµРІРѕРґ РЅР° РїСЂРѕРµРєС‚Рµ** = РїРѕСЃР»РµРґРѕРІР°С‚РµР»СЊРЅС‹Р№ РІС‹Р·РѕРІ **В«РїРµСЂРµРІРѕРґ РѕРґРЅРѕР№ РіР»Р°РІС‹В»** (РѕРґРёРЅ СЂР°Р· РёР»Рё С†РёРєР»РёС‡РЅРѕ РїРѕ СЃРїРёСЃРєСѓ РіР»Р°РІ).

- Р•РґРёРЅСЃС‚РІРµРЅРЅР°СЏ СЃРµСЂРІРµСЂРЅР°СЏ РѕРїРµСЂР°С†РёСЏ: `POST /api/projects/:projectId/chapters/:chapterId/translate` СЃ РѕРїС†РёСЏРјРё РѕР±СЉС‘РјР° (РІСЃСЏ РіР»Р°РІР° / С‚РѕР»СЊРєРѕ РїСѓСЃС‚С‹Рµ РїР°СЂР°РіСЂР°С„С‹ / РІС‹Р±СЂР°РЅРЅС‹Рµ РїР°СЂР°РіСЂР°С„С‹).

### 1.2 Р“Р»РѕСЃСЃР°СЂРёР№ Рё Р°РЅР°Р»РёР· вЂ” РѕР±С‰РёРµ РЅР° РїСЂРѕРµРєС‚

- Р“Р»РѕСЃСЃР°СЂРёР№ Рё РєРѕРЅС‚РµРєСЃС‚ Р°РЅР°Р»РёР·Р° **РЅР°РєР°РїР»РёРІР°СЋС‚СЃСЏ РѕС‚ РіР»Р°РІС‹ Рє РіР»Р°РІРµ** РІ СЂР°РјРєР°С… РѕРґРЅРѕРіРѕ РїСЂРѕРµРєС‚Р°.

  - Р’ РґРІРёР¶РєРµ: Р°РіРµРЅС‚ РєСЌС€РёСЂСѓРµС‚СЃСЏ РїРѕ `projectId`, СЃС‚Р°РґРёСЏ Р°РЅР°Р»РёР·Р° РґРѕРїРѕР»РЅСЏРµС‚ РєРѕРЅС‚РµРєСЃС‚, СЃР»РµРґСѓСЋС‰РёР№ РІС‹Р·РѕРІ pipeline РґР»СЏ РґСЂСѓРіРѕР№ РіР»Р°РІС‹ РІРёРґРёС‚ СѓР¶Рµ РЅР°РєРѕРїР»РµРЅРЅС‹Р№ РіР»РѕСЃСЃР°СЂРёР№.
- Р’ СЂРµС„Р°РєС‚РѕСЂРёРЅРіРµ РІР°Р¶РЅРѕ **РЅРµ СЂР°Р·РјР°Р·С‹РІР°С‚СЊ** Р»РѕРіРёРєСѓ РіР»РѕСЃСЃР°СЂРёСЏ РїРѕ СЂР°Р·РЅС‹Рј РјРµСЃС‚Р°Рј: РѕРґРёРЅ РёСЃС‚РѕС‡РЅРёРє РїСЂР°РІРґС‹ вЂ” Р°РіРµРЅС‚ + Р‘Р” РїСЂРѕРµРєС‚Р°; РІСЃРµ РІС‹Р·РѕРІС‹ РїРµСЂРµРІРѕРґР° РіР»Р°РІС‹ РёРґСѓС‚ С‡РµСЂРµР· РѕРґРёРЅ Рё С‚РѕС‚ Р¶Рµ РїСѓС‚СЊ, РіРґРµ Р°РіРµРЅС‚ РїРѕР»СѓС‡Р°РµС‚ Р°РєС‚СѓР°Р»СЊРЅС‹Р№ `project` (СЃ РіР»РѕСЃСЃР°СЂРёРµРј).

### 1.3 РџРµСЂРµРІРѕРґ РїР°СЂР°РіСЂР°С„РѕРІ СЃРѕС…СЂР°РЅСЏРµРј

  3. **Р’С‹Р±СЂР°РЅРЅС‹Рµ РїР°СЂР°РіСЂР°С„С‹** вЂ” `paragraphIds: string[]`; РїРµСЂРµРІРѕРґРёРј С‚РѕР»СЊРєРѕ СѓРєР°Р·Р°РЅРЅС‹Рµ РїР°СЂР°РіСЂР°С„С‹.
- Р­С‚Рѕ СѓР¶Рµ СЂРµР°Р»РёР·РѕРІР°РЅРѕ РІ API Рё РІ `performTranslation`; РІ РЅРѕРІРѕР№ Р°СЂС…РёС‚РµРєС‚СѓСЂРµ СЌС‚Рѕ РѕСЃС‚Р°С‘С‚СЃСЏ РµРґРёРЅСЃС‚РІРµРЅРЅС‹Рј СЃРїРѕСЃРѕР±РѕРј В«С‡Р°СЃС‚РёС‡РЅРѕРіРѕВ» РїРµСЂРµРІРѕРґР° (РЅР° СѓСЂРѕРІРЅРµ РїР°СЂР°РіСЂР°С„РѕРІ РѕРґРЅРѕР№ РіР»Р°РІС‹).

### 1.4 РћРґРЅРѕ РјРµСЃС‚Рѕ РІС…РѕРґР° РЅР° РєР»РёРµРЅС‚Рµ

- Р’СЃСЏ РёРЅРёС†РёР°С†РёСЏ РїРµСЂРµРІРѕРґР° РЅР° РєР»РёРµРЅС‚Рµ РёРґС‘С‚ С‡РµСЂРµР· **РѕРґРёРЅ СЃР»РѕР№** (СЃРµСЂРІРёСЃ РёР»Рё С…СѓРє):

- РЎС‚СЂР°РЅРёС†Р° РїСЂРѕРµРєС‚Р° Рё СЃС‚СЂР°РЅРёС†Р° РіР»Р°РІС‹ **РЅРµ** РґСѓР±Р»РёСЂСѓСЋС‚ Р»РѕРіРёРєСѓ: РѕРЅРё С‚РѕР»СЊРєРѕ РІС‹Р·С‹РІР°СЋС‚ СЌС‚РѕС‚ РѕР±С‰РёР№ СЃР»РѕР№ СЃ СЂР°Р·РЅС‹РјРё РїР°СЂР°РјРµС‚СЂР°РјРё (РѕРґРЅР° РіР»Р°РІР° / СЃРїРёСЃРѕРє РіР»Р°РІ, РѕРїС†РёРё РѕР±СЉС‘РјР°).

## 2. РЎРµСЂРІРµСЂРЅР°СЏ Р°СЂС…РёС‚РµРєС‚СѓСЂР°

### 2.1 Р•РґРёРЅСЃС‚РІРµРЅРЅС‹Р№ СЌРЅРґРїРѕРёРЅС‚ РїРµСЂРµРІРѕРґР°

```
POST /api/projects/:projectId/chapters/:chapterId/translate
Body: {
  translateOnlyEmpty?: boolean;
  paragraphIds?: string[];
  stages?: 'analysis' | 'translation' | 'editing' | 'all';  // РїРѕ СѓРјРѕР»С‡Р°РЅРёСЋ 'all'
}
```

- **РћР±СЉС‘Рј:** Р±РµР· С‚РµР»Р° РёР»Рё `{}` вЂ” РІСЃСЏ РіР»Р°РІР°; `translateOnlyEmpty: true` вЂ” С‚РѕР»СЊРєРѕ РїСѓСЃС‚С‹Рµ РїР°СЂР°РіСЂР°С„С‹; `paragraphIds: string[]` вЂ” С‚РѕР»СЊРєРѕ СѓРєР°Р·Р°РЅРЅС‹Рµ РїР°СЂР°РіСЂР°С„С‹.
- **РЎС‚Р°РґРёРё:** `stages: 'all'` (РїРѕ СѓРјРѕР»С‡Р°РЅРёСЋ) вЂ” Р°РЅР°Р»РёР· в†’ РїРµСЂРµРІРѕРґ в†’ СЂРµРґР°РєС‚СѓСЂР°; `'analysis'` вЂ” С‚РѕР»СЊРєРѕ Р°РЅР°Р»РёР· (РѕР±РЅРѕРІР»РµРЅРёРµ РіР»РѕСЃСЃР°СЂРёСЏ); `'translation'` вЂ” С‚РѕР»СЊРєРѕ РїРµСЂРµРІРѕРґ; `'editing'` вЂ” С‚РѕР»СЊРєРѕ СЂРµРґР°РєС‚СѓСЂР° СѓР¶Рµ РїРµСЂРµРІРµРґС‘РЅРЅРѕРіРѕ С‚РµРєСЃС‚Р°.

Р’РЅСѓС‚СЂРё СЂРѕСѓС‚Р°:

1. Р—Р°РіСЂСѓР·РєР° РїСЂРѕРµРєС‚Р° Рё РіР»Р°РІС‹, СЂР°СЃС‡С‘С‚ РѕР±СЉС‘РјР° С‚РµРєСЃС‚Р° (РІСЃСЏ РіР»Р°РІР° / РїСѓСЃС‚С‹Рµ / РІС‹Р±СЂР°РЅРЅС‹Рµ) РґР»СЏ РѕС†РµРЅРєРё С‚РѕРєРµРЅРѕРІ.
2. РџСЂРѕРІРµСЂРєР° Р»РёРјРёС‚Р° С‚РѕРєРµРЅРѕРІ: `estimateTokensForTranslation(textLength, options)` Рё `checkTokenLimit(userId, token, estimatedTokens)`; РїСЂРё РїСЂРµРІС‹С€РµРЅРёРё вЂ” 429, СЃС‚Р°С‚СѓСЃ РіР»Р°РІС‹ РІ `pending`.
3. РћР±РЅРѕРІР»РµРЅРёРµ СЃС‚Р°С‚СѓСЃР° РіР»Р°РІС‹ РІ `translating`.
4. Р—Р°РїСѓСЃРє `performTranslation(projectId, chapterId, chapter, project, ..., translateOnlyEmpty, token, userId, paragraphIds)` **РІ С„РѕРЅРµ** (РЅРµ Р±Р»РѕРєРёСЂРѕРІР°С‚СЊ РѕС‚РІРµС‚).
5. РћС‚РІРµС‚ РєР»РёРµРЅС‚Сѓ: `{ status: 'started', chapterId }`.

РўРµРєСѓС‰Р°СЏ СЂРµР°Р»РёР·Р°С†РёСЏ СѓР¶Рµ СЃРѕРѕС‚РІРµС‚СЃС‚РІСѓРµС‚ СЌС‚РѕРјСѓ; РїСЂРё СЂРµС„Р°РєС‚РѕСЂРёРЅРіРµ РґРѕСЃС‚Р°С‚РѕС‡РЅРѕ СЃРѕС…СЂР°РЅРёС‚СЊ РѕРґРёРЅ СЂРѕСѓС‚ Рё РІС‹РЅРµСЃС‚Рё РїСЂРё РЅРµРѕР±С…РѕРґРёРјРѕСЃС‚Рё СЂР°СЃС‡С‘С‚ С‚РµРєСЃС‚Р°/РѕРїС†РёР№ РІ РјР°Р»РµРЅСЊРєРёРµ С„СѓРЅРєС†РёРё.

### 2.2 performTranslation (Р»РѕРіРёРєР° РїРµСЂРµРІРѕРґР° РѕРґРЅРѕР№ РіР»Р°РІС‹)

- Р’С…РѕРґ: РїСЂРѕРµРєС‚, РіР»Р°РІР°, С„Р»Р°РіРё `translateOnlyEmpty`, `paragraphIds`, **`stages: 'analysis'|'translation'|'editing'|'all'`**, С‚РѕРєРµРЅ, userId.
- РћРїСЂРµРґРµР»РµРЅРёРµ РѕР±СЉС‘РјР°: РєР°РєРёРµ РїР°СЂР°РіСЂР°С„С‹ РїРµСЂРµРІРѕРґРёРј (РІСЃРµ / РїСѓСЃС‚С‹Рµ / РїРѕ ID), СЃР±РѕСЂ С‚РµРєСЃС‚Р° РґР»СЏ pipeline.
- Р’С‹Р·РѕРІ РґРІРёР¶РєР° РІ Р·Р°РІРёСЃРёРјРѕСЃС‚Рё РѕС‚ `stages`:

Р“Р»РѕСЃСЃР°СЂРёР№ РјРµР¶РґСѓ РіР»Р°РІР°РјРё Р°РІС‚РѕРјР°С‚РёС‡РµСЃРєРё РЅР°РєР°РїР»РёРІР°РµС‚СЃСЏ (СЃРµСЂРІРµСЂ РїСЂРё РєР°Р¶РґРѕРј Р·Р°РїСЂРѕСЃРµ РїРѕРґРіСЂСѓР¶Р°РµС‚ РїСЂРѕРµРєС‚ СЃ Р‘Р”).

3. **РђРІС‚РѕСЂ СЃР°Рј СЂРµРґР°РєС‚РёСЂСѓРµС‚ РёР»Рё РїРµСЂРµРІРѕРґРёС‚ С‡Р°СЃС‚СЊ**  
   Р РµРґР°РєС‚РёСЂСѓРµС‚ РїР°СЂР°РіСЂР°С„С‹ РІСЂСѓС‡РЅСѓСЋ; РґР»СЏ РѕСЃС‚Р°РІС€РёС…СЃСЏ РїСѓСЃС‚С‹С… РёСЃРїРѕР»СЊР·СѓРµС‚ В«РџРµСЂРµРІРµСЃС‚Рё РїСѓСЃС‚С‹РµВ». ### 5.2 Р Р°Р·РґРµР»РµРЅРёРµ СЃС‚РµР№РґР¶РµР№: Р°РЅР°Р»РёР·, РїРµСЂРµРІРѕРґ, СЂРµРґР°РєС‚СѓСЂР°

**РРґРµСЏ:** СЃС‚Р°РґРёРё РЅРµ РІРєР»СЋС‡Р°СЋС‚СЃСЏ/РІС‹РєР»СЋС‡Р°СЋС‚СЃСЏ РІ РЅР°СЃС‚СЂРѕР№РєР°С…, Р° **РІС‹Р·С‹РІР°СЋС‚СЃСЏ СЏРІРЅРѕ** СЃ UI. РџРѕР»СЊР·РѕРІР°С‚РµР»СЊ СЃР°Рј СЂРµС€Р°РµС‚, С‡С‚Рѕ Р·Р°РїСѓСЃС‚РёС‚СЊ: С‚РѕР»СЊРєРѕ Р°РЅР°Р»РёР· (РѕР±РЅРѕРІРёС‚СЊ РіР»РѕСЃСЃР°СЂРёР№), С‚РѕР»СЊРєРѕ РїРµСЂРµРІРѕРґ, С‚РѕР»СЊРєРѕ СЂРµРґР°РєС‚СѓСЂСѓ РёР»Рё РІСЃРµ С‚СЂРё РїРѕРґСЂСЏРґ.

- **РќР°СЃС‚СЂРѕР№РєРё РїСЂРѕРµРєС‚Р°:** СѓР±СЂР°С‚СЊ Р±Р»РѕРє В«Р­С‚Р°РїС‹ РїРµСЂРµРІРѕРґР°В» (РїРµСЂРµРєР»СЋС‡Р°С‚РµР»Рё enableAnalysis / enableEditing). РћСЃС‚Р°РІРёС‚СЊ: РјРѕРґРµР»Рё РїРѕ СЃС‚Р°РґРёСЏРј (analysis, translation, editing), РєСЂРµР°С‚РёРІРЅРѕСЃС‚СЊ (temperature), СЂРµР¶РёРј В«С‚РѕР»СЊРєРѕ РѕСЂРёРіРёРЅР°Р»В». РўРѕ РµСЃС‚СЊ РІ РЅР°СЃС‚СЂРѕР№РєР°С… вЂ” С‚РѕР»СЊРєРѕ **РєР°РєРёРµ РЅРµР№СЂРѕРЅРєРё Рё РїР°СЂР°РјРµС‚СЂС‹** РёСЃРїРѕР»СЊР·РѕРІР°С‚СЊ, Р° РЅРµ РєР°РєРёРµ СЃС‚Р°РґРёРё В«РІРєР»СЋС‡РµРЅС‹В».
- **API:** РѕРґРёРЅ Рё С‚РѕС‚ Р¶Рµ СЌРЅРґРїРѕРёРЅС‚ РїРµСЂРµРІРѕРґР° РіР»Р°РІС‹ СЂР°СЃС€РёСЂРёС‚СЊ С‚РµР»РѕРј:
  - `stages?: 'analysis' | 'translation' | 'editing' | 'all'` (РёР»Рё РјР°СЃСЃРёРІ `runStages: ('analysis'|'translation'|'editing')[]`).
  - РџРѕ СѓРјРѕР»С‡Р°РЅРёСЋ `stages: 'all'` вЂ” С‚РµРєСѓС‰РµРµ РїРѕРІРµРґРµРЅРёРµ (Р°РЅР°Р»РёР· в†’ РїРµСЂРµРІРѕРґ в†’ СЂРµРґР°РєС‚СѓСЂР°).
  - `stages: 'analysis'` вЂ” С‚РѕР»СЊРєРѕ СЃС‚Р°РґРёСЏ Р°РЅР°Р»РёР·Р°, РѕР±РЅРѕРІР»РµРЅРёРµ РіР»РѕСЃСЃР°СЂРёСЏ РїРѕ РїСЂРѕРµРєС‚Сѓ, РіР»Р°РІР° РЅРµ РїРµСЂРµРІРѕРґРёС‚СЃСЏ (РёР»Рё С‚РѕР»СЊРєРѕ СЃСѓС‰РЅРѕСЃС‚Рё СЃРѕС…СЂР°РЅСЏСЋС‚СЃСЏ).
  - `stages: 'translation'` вЂ” С‚РѕР»СЊРєРѕ РїРµСЂРµРІРѕРґ (РіР»РѕСЃСЃР°СЂРёР№ СѓР¶Рµ РµСЃС‚СЊ; РјРѕР¶РЅРѕ Р·Р°РїСѓСЃРєР°С‚СЊ РїРѕСЃР»Рµ Р°РЅР°Р»РёР·Р° РёР»Рё РµСЃР»Рё РіР»РѕСЃСЃР°СЂРёР№ Р·Р°РїРѕР»РЅРµРЅ РІСЂСѓС‡РЅСѓСЋ).
  - `stages: 'editing'` вЂ” С‚РѕР»СЊРєРѕ СЂРµРґР°РєС‚СѓСЂР° СѓР¶Рµ РїРµСЂРµРІРµРґС‘РЅРЅРѕРіРѕ С‚РµРєСЃС‚Р° РіР»Р°РІС‹.
- **РћС†РµРЅРєР° С‚РѕРєРµРЅРѕРІ:** РїСЂРё РІС‹Р·РѕРІРµ РѕРґРЅРѕР№ СЃС‚Р°РґРёРё СЃС‡РёС‚Р°РµРј С‚РѕР»СЊРєРѕ РµС‘ (analysis / translation / editing). Р¤РѕСЂРјСѓР»Р°: С‚Рµ Р¶Рµ РєРѕСЌС„С„РёС†РёРµРЅС‚С‹ РЅР° 10k СЃРёРјРІРѕР»РѕРІ РїРѕ СЃС‚Р°РґРёСЏРј; РїСЂРё `stages: 'all'` вЂ” СЃСѓРјРјР° С‚СЂС‘С… СЃС‚Р°РґРёР№. - **Р¤Р»РѕСѓ:** Р°РІС‚РѕСЂ РјРѕР¶РµС‚ СЃРЅР°С‡Р°Р»Р° Р·Р°РїСѓСЃС‚РёС‚СЊ В«РђРЅР°Р»РёР·В» РїРѕ РіР»Р°РІРµ (РёР»Рё РїРѕ РЅРµСЃРєРѕР»СЊРєРёРј РіР»Р°РІР°Рј РїРѕРґСЂСЏРґ), РїРѕС‚РѕРј В«РџРµСЂРµРІРѕРґВ», РїРѕС‚РѕРј РїСЂРё Р¶РµР»Р°РЅРёРё В«Р РµРґР°РєС‚СѓСЂР°В». РР»Рё СЃСЂР°Р·Сѓ В«Р’СЃС‘В» (Р°РЅР°Р»РёР· + РїРµСЂРµРІРѕРґ + СЂРµРґР°РєС‚СѓСЂР°). Р“Р»РѕСЃСЃР°СЂРёР№ РїРѕ-РїСЂРµР¶РЅРµРјСѓ РѕР±С‰РёР№ РЅР° РїСЂРѕРµРєС‚ Рё РЅР°РєР°РїР»РёРІР°РµС‚СЃСЏ РїСЂРё Р°РЅР°Р»РёР·Рµ.

### 5.3 Р”РёР·Р°Р№РЅ РєР»РёРµРЅС‚СЃРєРѕР№ С‡Р°СЃС‚Рё: РіРґРµ Рё РєР°Рє РІС‹Р·С‹РІР°С‚СЊ РїРµСЂРµРІРѕРґ

**РџСЂРѕР±Р»РµРјР°:** РЅР° РіР»Р°РІРµ СЃРµР№С‡Р°СЃ РјРЅРѕРіРѕ РјРµСЃС‚Р° Р·Р°РЅРёРјР°СЋС‚ РєРЅРѕРїРєРё (СЃС‚Р°С‚СѓСЃ, С‡РёС‚Р°С‚СЊ, РѕРґРѕР±СЂРёС‚СЊ РІСЃРµ, РІС‹Р±СЂР°С‚СЊ РІСЃРµ/СЃРЅСЏС‚СЊ, РїРµСЂРµРІРµСЃС‚Рё РїСѓСЃС‚С‹Рµ, РїРµСЂРµРІРµСЃС‚Рё РІС‹Р±СЂР°РЅРЅС‹Рµ, РѕС‚РјРµРЅРёС‚СЊ, РїРµСЂРµРІРµСЃС‚Рё/РїРѕРІС‚РѕСЂРёС‚СЊ/РµС‰С‘ СЂР°Р·, РЅР°СЃС‚СЂРѕР№РєРё). РќСѓР¶РЅРѕ СЃРѕРєСЂР°С‚РёС‚СЊ РІРёР·СѓР°Р»СЊРЅС‹Р№ С€СѓРј Рё СЏРІРЅРѕ СЂР°Р·РґРµР»РёС‚СЊ РІС‹Р·РѕРІС‹ РїРѕ СЃС‚Р°РґРёСЏРј.

**РџСЂРµРґР»Р°РіР°РµРјР°СЏ СЃС…РµРјР°:**

1. **РЁР°РїРєР° РіР»Р°РІС‹ (ChapterHeader) вЂ” РєРѕРјРїР°РєС‚РЅРѕ**
   - РћСЃС‚Р°РІР»СЏРµРј РІ РѕРґРЅРѕР№ СЃС‚СЂРѕРєРµ: РЅР°РІРёРіР°С†РёСЏ (в—Ђ РЅР°Р·РІР°РЅРёРµ в–¶), СЃС‚Р°С‚СѓСЃ РіР»Р°РІС‹, РєРЅРѕРїРєРё **В«рџ“– Р§РёС‚Р°С‚СЊВ»** Рё **В«вљ™пёЏВ»** (РЅР°СЃС‚СЂРѕР№РєРё РѕС‚РѕР±СЂР°Р¶РµРЅРёСЏ).
