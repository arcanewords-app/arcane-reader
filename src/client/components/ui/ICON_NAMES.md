# Canonical Material Symbol Names

Reference for control icons in Arcane Reader. Use via `@src/client/components/ui/Icon.tsx` (Outlined, loaded in `index.html`).

## Navigation

`arrow_back`, `arrow_downward`, `arrow_upward`, `chevron_left`, `chevron_right`, `close`, `language`, `more_vert`

## Reader

`menu_book`, `toc`, `share`, `settings`, `content_copy`, `search`

## Translation

`translate`, `manage_search`, `edit`, `auto_awesome`, `check_circle`, `done_all`, `skip_next`

## Glossary / actions

`person`, `place`, `delete`, `upload_file`, `image`, `warning`, `error`, `schedule`

## Legacy emoji → icon

Prefer Material Symbols over emoji for controls: `arrow_back`, `menu_book`, `translate`, `delete`, etc. Reject new emoji control icons in PR.
